Utilizando os conceitos visto em aula, desenvolver um programa em Java ou C que permita gerenciar dispositivos (luzes, tomadas, TV, ar-condicionado) de uma casa. 

Deverão ser criadas as estruturas que representem dispositivos, cômodos e casa. Para dispositivos deverá ser possível inserir o nome (abajur, espelho, tv, central, etc), tipo (luz, tomada, ar-condicionada, TV, etc) e estado (Ligado/Desligado).

Na estrutura “cômodo” deverão permitir a captura de nome (sala, cozinha, etc) e uma lista de dispositivos onde deverão de inserir os dispositivos que se encontram no respectivo cômodo.

E finalmente na estrutura casa deverá representar uma lista de cômodos com seus respectivos dispositivos.

O programa deverá permitir todas as operações de lista ou qualquer outra estrutura (criar, inserir, remover, listar, consultar e modificar), uma função que realize a mudança de estado (ligado/desligado) de um dispositivo enviado por parâmetro e uma função para listar todos os dispositivos ligados dentro da casa. Na lista deverá aparecer o nome do cômodo e nome do dispositivo.
